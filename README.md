## 🌟 Features

- **Premium UI/UX:** Glassmorphism, subtle cinematic background animations, and custom 3D hover effects.
- **High Performance:** 60 FPS animations via `requestAnimationFrame` and hardware-accelerated CSS transforms. Lazy loaded sections for instant load times.
- **Fully Responsive:** Flawless scaling from ultra-wide desktops to small mobile devices.
- **SEO Optimized:** Perfect semantic HTML, JSON-LD structured data, robust metadata, and OpenGraph social preview tags.
- **Interactive:** Custom GPU-accelerated cursor, intersection observers for scroll reveals, and magnetic buttons.

## 🚀 Deployment Instructions

This project is a static site and requires zero build steps! It is incredibly easy to deploy:

### Vercel / Netlify
1. Connect your GitHub repository to Vercel/Netlify.
2. Ensure the build command is **empty**.
3. Set the output directory to `.` (the root).
4. Deploy!

### GitHub Pages
1. Go to your repository **Settings** > **Pages**.
2. Select the `main` branch and `/root` directory.
3. Save, and your portfolio will be live at `https://<username>.github.io/<repo-name>/`.
